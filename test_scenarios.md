# тесткейсы

## TC-01. Загрузка страницы результатов
**Шаги:**
1. Перейти на страницу результатов Amsterdam.
2. Дождаться отображения заголовка и списка отелей.

**Ожидаемый результат:** заголовок страницы виден, есть как минимум одна ссылка "Check availability".

## TC-02. Открытие карточки отеля

1. Нажать первую ссылку "Check availability".
2. Дождаться страницы отеля.

**Ожидаемый результат:** открыт профиль отеля, виден заголовок и кнопка/ссылка бронирования.

## TC-03. Открытие карты
**Шаги:**
1. Нажать "Show hotels on map".

**Ожидаемый результат:** URL содержит map, страница карты открыта.

## TC-04. Сортировка результатов
**Предусловие:** открыта страница результатов.
**Шаги:**
1. Нажать фильтр "Оценка + кол-во отзывов".
**Ожидаемый результат:** список результатов остаётся доступным, отели отсортированы в надлежащем порядке по рейтингу (сначала с хорошим).

## TC-05. Фильтрация по звёздам
**Предусловие:** открыта страница результатов.
**Шаги:**
1. Нажать "5 stars".
**Ожидаемый результат:** страница остаётся доступной и результаты обновляются.
